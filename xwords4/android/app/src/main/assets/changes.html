CrossWords 4.4.169 release

This release fixes a wordlist filter crash and improves tile dragging

Please take this survey to help determine what gets done next in CrossWords.

New with this release

  • Fix crash applying certain complicated wordlist filters
  • When a tile drag ends where it can't be dropped, drop it on last good location instead of where it started
  • Prevent deleting the default games group
  • Show orphaned games in new Recovered group
  • Make long-tap on wordlist entry copy word to clipboard
