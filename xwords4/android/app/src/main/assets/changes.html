CrossWords 4.4.189 release

This release improves battery life, adds archiving, and makes it easier to support play in new languages.

Please take this survey to help determine what gets done next in CrossWords.

New with this release

  • Better support for wordlists in new languages
  • Add ability to archive settings and games or to migrate them to a new device
  • Improve efficiency of standalone games
  • A few UI cleanups
  • Translations via Weblate in Brazilian Portuguese, Chinese, German, Hungarian (new!), and Portuguese

Coming soon

  • Support duplicate-style play (popular in France)
  • Copy Tutanota's push-notification technique
  • Improve move-via-NFC

Please let me know (email xwords@eehouse.org) what features you’d most like to see or if you’re experiencing problems. And if you do get a crash, please let me know!

Thanks!
