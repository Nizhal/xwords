CrossWords 4.4.198 release

This release fixes a few bugs.

Please take this survey to help determine what gets done next in CrossWords.

New with this release

  • Fix bug rematching standalone games with non-default wordlists
  • Remove 255-char limit on chat messages
  • On Android 9 only, warn that restore doesn't work
  • Include translations in Brazilian Portuguese, German, Portuguese and Turkish (via Weblate)

Coming soon

  • Support duplicate-style play (popular in France)
  • Copy Tutanota's push-notification technique
  • Improve move-via-NFC

Please let me know (email xwords@eehouse.org) what features you’d most like to see or if you’re experiencing problems. And if you do get a crash, please let me know!
