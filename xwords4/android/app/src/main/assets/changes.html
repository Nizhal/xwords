CrossWords 4.4.180 release

This release adds "dark mode"

Please take this survey to help determine what gets done next in CrossWords.

New with this release

  • Add "Dark" mode, off by default unless "dark theme" is enabled in Android Settings
  • In App Settings, replace buttons with a popup menu
  • Add option to share color themes (dark or light) via URLs on clipboard
  • Fix bug preventing trading when 8 or 9 tiles left in pool
  • Fix bug selecting 9th tile in 9-tile tray
  • Fix bug not reporting deleted games when using SMS Data
  • New translations in Chinese (Simplified), French, Indonesian, Portuguese, and Portuguese (Brazil)

(The full changelog is here.)

Coming soon

  • Phase out play-by-relay (in favor of MQTT)
  • Improve move-via-NFC
  • Support duplicate-style play (popular in France)
  • Improve play-by-data-sms workaround using NBSProxy

Please let me know (email xwords@eehouse.org) what features you’d most like to see or if you’re experiencing problems. And if you do get a crash, please let me know!
