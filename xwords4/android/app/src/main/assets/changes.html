CrossWords 4.4.164 release

This release includes improvements to the UI and to networking.

Please take this survey to help determine what gets done next in CrossWords.

New with this release

  • Improve connections and communication between games started without an invitation (but: you should still use an invitation when you can)
  • Indicate selected games and groups with a checkmark
  • Move games list scrollbar to the right side
  • Tweak layout of wordlist filter
  • Improve player-config dialog
  • Fix occasional problems remembering game groups ordering
  • Fix crashes reported to Google (thanks, reporters!)
  • Fix access to changelog (link just below)

(The full changelog is here.)

Coming soon

  • Improve initial game connectivity
  • Improve move-via-NFC
  • Support duplicate-style play (popular in France)
  • Improve play-by-data-sms workaround using NBSProxy

Please let me know (email xwords@eehouse.org) what's broken and what features you'd most like to see. And if you do get a crash (on a Play Store release), please let your device report it to Google so I can see it!
