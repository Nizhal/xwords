CrossWords 4.4.194 release

This release improves inviting/rematching and move transmission, and fixes false remote-game-deleted reports

Please take this survey to help determine what gets done next in CrossWords.

New with this release

  • Improve move transmission when devices aren't online at the same time
  • Make invitations more reliable for Known Players and rematches
  • Fix Bluetooth-connected devices lying about games being deleted
  • Improve Game Configuration dialog
  • Improve Game Over dialog
  • Improve permissions for latest Android OS version
  • Fix sharing of color themes
  • Fix Robot to not play a blank for 0 points (duh)
  • Fix accepting invitations on Android 12
  • Fix wordlist upgrading
  • Include translations in Brazilian Portuguese, Chinese, Czech, German, Hungarian, Indonesian, Portuguese, Russian, Simplified Chinese and Spanish (via Weblate)

Coming soon

  • Improve invitation process
  • Support duplicate-style play (popular in France)
  • Copy Tutanota's push-notification technique
  • Improve move-via-NFC

Please let me know (email xwords@eehouse.org) what features you’d most like to see or if you’re experiencing problems. And if you do get a crash, please let me know!
